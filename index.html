Projects

While a lot of my ideas end up into the trashpile, here are some somewhat-working projects that I have completed.

  • Fast Startup - Modification for Lethal Company that skips the unnecessary intro videos.
  • Gunpowder Go Boom - Minecraft server plugin which adds a custom recipe with gunpowder in a Furnace that makes an explosion.
  • SimPresence - Microsoft Flight Simulator integration for Discord that shows your Altitude, Airspeed and Location.
  • CLI Args Logger - Small utility program that logs to a file with the arguments that were called to run the program.
